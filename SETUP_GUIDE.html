1
Telegram Bot Banao (BotFather)

Telegram par @BotFather search karo aur yeh steps follow karo:

/start
/newbot
Name: InstaGrab Bot
Username: instagrab_mybot (unique hona chahiye)
# BotFather aapko ek TOKEN dega — save kar lo!
# Example: 7123456789:AAHxxxxxxxxxxxxxxxxxxxxxxxx
⚠️ Token kisi ke saath share mat karo — yeh aapka bot ka password hai!

2
Code GitHub par Upload Karo

Teen files GitHub par push karni hain:

ℹ️ bot.py, requirements.txt, aur railway.toml — ye teeno files download karo (upar buttons se) aur GitHub repo mein upload karo.
git init
git add .
git commit -m "InstaGrab Bot initial commit"
git remote add origin https://github.com/YOURUSERNAME/instabot.git
git push -u origin main

Ya GitHub.com par directly new repo banao aur files drag-drop karo.

3
Railway par Deploy Karo

Railway.app bilkul free hai — credit card bhi nahi chahiye starter plan ke liye.

  1. railway.app par jao → GitHub se Login karo
  2. New Project → Deploy from GitHub Repo
  3. Apna instabot repo select karo
  4. Project create hone ke baad Variables tab mein jao
  5. Neeche diye environment variables add karo
  6. Deploy button dabao — 2 minute mein live!

🔧 Problems? Solutions

Bot respond nahi kar raha?
Railway dashboard → Logs check karo. BOT_TOKEN sahi set hai?

Download fail ho raha hai?
Post private ho sakti hai. Public posts hi download hoti hain without login.

Instagram block kar raha hai?
IG_USERNAME aur IG_PASSWORD set karo. Ya thodi der baad try karo.

Video too large?
50MB se badi videos Telegram mein nahi jaati (bot limit). Reels mostly okay hoti hain.
